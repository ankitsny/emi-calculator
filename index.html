↳ Part Payments / Prepayments

▾

One-time lump-sum payments applied before that month's EMI. Free prepayment is allowed on most floating-rate home loans. Choose Standard only, OD only, or both.

↳ OD Balance — Period Overrides

▾

Override the default OD balance for specific periods. Model salary hikes, annual bonus months, or lean periods. Ranges can overlap — last matching range wins. Gaps between ranges = ₹0 parked. Till End appears only on the last row.

When & Why to Use State Files

A state file is a snapshot of everything you've entered: principal, rates, tenure, part payments, OD periods, and rate overrides — saved as a readable .json file on your device.

  • Share scenarios — send a colleague or advisor your exact scenario without screen-sharing
  • Compare alternatives — save "Scenario A: 20yr tenure" and "Scenario B: 15yr tenure + prepayment" as separate files
  • Resume later — browser storage clears on some devices (incognito, iOS Safari); a state file is permanent
  • Audit trail — keep a dated file for each time your loan rate changes, so you have a history
  • Team review — share a JSON file with your CA or broker so they can load and verify your assumptions

Use Export state after you run Calculate (export is disabled until a report is shown). Import state replaces your current inputs.
